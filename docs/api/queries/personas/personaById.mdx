# personaById

Retrieves the details of a single persona.

## Parameters

<ParamFormat param="personaId" type="ID!" />

The ID of the persona to be queried. Personas are stored in 
alphabetical order, with their ID corresponding to their 
position within that order.

## Query

```graphql title="QUERY" showLineNumbers
query personaById($personaId: ID!) {
  personaById(personaId: $personaId) {
    ...PersonaFragment
  }
}
```
