# personaRecipesById

Retrieves all potential fusion combinations to create a given persona.

## Parameters

<ParamFormat param="personaId" type="ID!" />

The ID of the persona to be queried. Personas are stored in 
alphabetical order, with their ID corresponding to their position 
within that order.

<ParamFormat param="dlc" type="Boolean!" />

<DlcDescription />

## Query

```graphql title="QUERY" showLineNumbers
query personaRecipesById(
  $personaId: ID!, 
  $dlc: Boolean!
) {
  personaRecipesById(
    personaId: $personaId, 
    dlc: $dlc
  ) {
    ...PersonaFragment
  }
}
```

{
  "personaId": "78",
  "dlc": true
}
