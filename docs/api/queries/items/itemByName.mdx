# itemByName

Retrieves the details of all items that match the parameters.

## Parameters

<ParamFormat param="name" type="String!" />

The name of the item to be queried. 
Partial values are accepted and all items that match the
parameter value will be queried. The parameter is case insensitive

## Query

```graphql title="QUERY" showLineNumbers
query itemByName($name: String!) {
  itemByName(name: $name) {
    itemId,
    name,
    type,
    description
  }
}
```

```jsx title="VARIABLES" showLineNumbers
{
  "name": "Paradise Lost"
}
```
