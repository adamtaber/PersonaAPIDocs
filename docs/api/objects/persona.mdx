# Persona

```graphql

  personaId: ID!,

  inheritanceType: String!,
```

## PersonaAffinities Object

```graphql
type PersonaStats {
  phys: String!,
  gun: String!,
  fire: String!,
  ice: String!,
  elec: String!,
  wind: String!,
  psy: String!,
  nuke: String!,
  bless: String!,
  curse: String!
}
```

### Fields

Each field's value is one of the abbreviation's in the table below or the
symbol "-", meaning that there is no affinity for that element.

| Abbreviation |  Affinity  |
| -----------  | ---------- |
| wk           | weak       |
| rs           | resist     |
| ab           | absorb     |
| rp           | repel      |
| nu           | null       |

In the game, "absorb" is instead listed as "drain" with an
abbreviation of "dr", and "resist" is listed as "strong" with an
abbreviation of "str". These incorrect abbreviations are used here
to match the most popular Persona 5 Royal resources found online.
